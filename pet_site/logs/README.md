# 📊 Logs 应用 - 宠物日志记录系统

## 📋 应用概述
`logs` 应用负责宠物日志系统，记录宠物的日常数据如体重、饮水量、食物摄入等，为长期健康分析提供数据基础。该应用支持AI服务集成，提供智能分析功能。

## 🏗️ 应用架构 
```
logs/
├── __init__.py              # 应用初始化
├── README.md               # 应用说明文档 (本文件)
├── admin.py                # 后台管理配置
├── apps.py                 # 应用配置
├── models.py               # 日志数据模型
├── views.py                # 日志视图逻辑
├── forms.py                # 日志表单
├── urls.py                 # URL路由配置
├── ai_service.py           # AI服务集成
├── migrations/             # 数据库迁移文件
└── templates/              # 模板文件
```

## 🗂️ 文件结构说明

### 📄 核心文件
- **`models.py`** - 日志数据模型定义
  - `PetLog` - 宠物日志主模型
  - `WeightLog` - 体重记录模型
  - `FeedingLog` - 喂食记录模型
  - `WaterIntakeLog` - 饮水记录模型
  - `ExerciseLog` - 运动记录模型

- **`views.py`** - 日志管理视图
  - 日志记录创建和编辑
  - 数据可视化展示
  - 趋势分析

- **`forms.py`** - 表单定义
  - `DailyLogForm` - 日常记录表单
  - `WeightTrackingForm` - 体重追踪表单
  - `FeedingRecordForm` - 喂食记录表单

- **`analytics.py`** - 数据分析模块
  - 健康趋势分析
  - 异常检测
  - 报告生成

### 📁 目录说明
- **`migrations/`** - 数据库迁移文件
- **`templates/logs/`** - 模板文件
- **`static/logs/`** - 静态资源（图表、样式）
- **`charts/`** - 图表生成模块
- **`reports/`** - 报告生成模块

## 🎯 主要功能
1. **日常记录** - 体重、饮食、运动等数据记录
2. **数据可视化** - 图表展示健康趋势
3. **异常提醒** - 数据异常自动提醒
4. **健康报告** - 定期生成健康分析报告
5. **数据导出** - 支持多种格式数据导出

## 📊 记录类型
```
LogTypes:
├── weight (体重记录)
├── feeding (喂食记录)
├── water_intake (饮水记录)
├── exercise (运动记录)
├── medication (用药记录)
├── behavior (行为记录)
└── health_check (健康检查)
```

## 🔗 关联关系
- 与 `pets` 应用：多对一关系（多条日志对应一个宠物）
- 与 `accounts` 应用：多对一关系（多条日志对应一个用户）
- 与 `consultations` 应用：数据支撑关系（为AI分析提供历史数据）
- 与 `core` 应用：继承基础模型和使用共享常量

- **预警系统** - 异常数据自动警报

## 📊 数据结构层次
```
PetLog (主日志表)
├── 基础信息：日期、宠物、创建者
├── 日常状态：心情、活跃度、食欲
├── 生理指标：体重、体温
└── 扩展记录：
    ├── FeedingLog (喂食记录)
    ├── ExerciseLog (运动记录)
    ├── HealthLog (健康记录)
    ├── MedicationLog (用药记录)
    └── LogAnalytics (分析数据)
```
